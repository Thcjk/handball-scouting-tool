# Spielanleitung (MVP)

## Start

1. Registriere dich mit E-Mail, Benutzername und Passwort
2. Gib deinem Königreich und Herrscher einen Namen
3. Du erhältst automatisch eine Provinz mit Burg, Dorf und Garnison

## Ressourcen

| Ressource | Verwendung                |
| --------- | ------------------------- |
| Gold      | Bau, Rekrutierung, Handel |
| Nahrung   | Einheiten-Unterhalt, Bau  |
| Holz      | Gebäude, Einheiten        |
| Stein     | Burg, Gebäude             |
| Eisen     | Waffen, Einheiten         |
| Einfluss  | Diplomatie (geplant)      |
| Ruhm      | Eroberungen               |

## Aktionen

### Burg ausbauen

Erhöht die Verteidigung der Provinz. Kostet Gold, Holz, Stein und Eisen.

### Gebäude bauen

- **Bauernhof** – Nahrungsproduktion
- **Mine** – Stein und Eisen
- **Sägewerk** – Holz
- **Kaserne** – Rekrutierungsbonus
- **Palisade** – Verteidigung

### Rekrutieren

Wähle Einheitentyp und Anzahl. Truppen werden der Garnison hinzugefügt.

### Armee bilden

Teilt die Garnison und erstellt eine bewegliche Armee (nicht-Garnison). Nur Armeen können angreifen.

### Angreifen

1. Wähle eine Nachbarprovinz auf der Karte
2. Klicke „Angreifen mit [Armee]"
3. Die Schlacht wird automatisch berechnet
4. Bei Sieg: Provinz wird erobert

## Einheiten

| Einheit            | Stärke        | Kosten     |
| ------------------ | ------------- | ---------- |
| Miliz              | Niedrig       | Günstig    |
| Speerträger        | Verteidigung  | Mittel     |
| Bogenschütze       | Angriff       | Mittel     |
| Schwertkämpfer     | Ausgewogen    | Hoch       |
| Armbrustschütze    | Hoher Angriff | Hoch       |
| Leichte Kavallerie | Schnell       | Teuer      |
| Schwere Kavallerie | Stark         | Sehr teuer |
| Ritter             | Elite         | Sehr teuer |

## Tipps

- Baue zuerst Ressourcen-Gebäude (Bauernhof, Mine, Sägewerk)
- Rüste deine Burg aus, bevor du angreifst
- Neutral Provinzen sind leichter zu erobern
- Armeen brauchen Nachbarprovinzen als Angriffsziel
