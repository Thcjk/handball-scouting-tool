# E-Mail-Benachrichtigungen abschalten

## Cursor (Agent / Cloud) – E-Mails stoppen

1. Öffne **Cursor** auf dem Computer
2. Gehe zu **Settings** (Zahnrad) → **Cursor Settings**
3. Suche **Notifications** oder **Email**
4. Deaktiviere:
   - **Email notifications for background agents**
   - **Email when agent completes**
   - Ähnliche Agent-/Cloud-Einträge

Alternativ: https://cursor.com/settings → Benachrichtigungen → E-Mail aus

> Das kann nur du in deinem Cursor-Konto ändern – der Agent hat keinen Zugriff darauf.

## GitHub (weniger Mails vom Repository)

1. https://github.com/settings/notifications
2. Bei **Actions** → Häkchen **Email** entfernen
3. Oder Repo **Kronenchronik** → **Watch** → **Custom** → Actions aus
